# CPU&GPU Architecture

>GPUs devote more transistors to **parallel data processing**, while CPUs spend more on **control logic and cache**. GPU 把更多晶体管用于**并行数据计算**，而 CPU 把更多资源用于**控制与缓存**。
>
>CPUs aim for **low-latency complex control**,  **低延迟的复杂控制**；
> while GPUs aim for **high-throughput parallel computation**. **高吞吐的并行计算**。
