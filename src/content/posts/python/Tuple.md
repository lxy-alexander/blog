## 📌 元组（Tuple）

A tuple is an ordered and immutable collection of elements.

Features:

-   Ordered
-   Immutable (cannot be changed after creation)
-   Can contain different data types

Example:

```python
t = (1, 2, 3)
```

Single-element tuple:

```python
t = (5,)
```

Tuples are immutable:

```python
t[0] = 10  # Error
```

------

## 🔹 Tuple vs List

|            | Tuple    | List     |
| ---------- | -------- | -------- |
| 符号       | `()`     | `[]`     |
| 是否可修改 | ❌ 不可改 | ✅ 可改   |
| 用途       | 固定数据 | 可变数据 |
